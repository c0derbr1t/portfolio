See the Project!

GitHub Repo

What is this?

This is a daily project where we were given instructions on GitHub to follow. The purpose was to use React and the Reducer Pattern to create a To-Do application. It was a precursor to actually learning react.

What kind of build was this?

Daily projects can be completed alone or by pair programming. We were encouraged to reach out to our Team Lead or other students for help as needed. We could not receive help with them. They reinforce the day's lesson, provide practice for the material, and expand your knowledge through self learning.

What did I do?

I used Create React App to initialize the project. Then I built a simple reducer and added initial state. I used that to render a list of todo's from a component. Then I built the form to add more todos to the list. I added a toggle to show whether the item was completed or not. I also created a function that filtered out and removed and completed items with a button click.
