What is this?

Sprint 2 in Unit 4 taught us about Relational Databases, Database Schema Design, Multi-Table Queries, and Data Modeling. This project tested our abilities with using those technologies to design and build an API with Data Persistence.

What kind of build was this?

Sprint Challenges are projects we completed alone within a 3 hours. We could not receive help with them. They act as tests of our knowledge and skill regarding the week's learning.

What did I do?

I was tasked with designing and building a Data Model and RESTful API that stores data into a Relational Database. I started from scratch and built out the project using Node.js, Knex, and Express according to the requirements specified. I added endpoints for full CRUD functionality.
