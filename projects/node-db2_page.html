What is this?

This is a daily project where we were given instructions on GitHub to follow. We were tasked with writing an API that could be used to manage Cars stored in a Relational Database.

This was a fun project. It's where the things I was learning started to come together. I wasn't just accessing information, I was BUILDING it!

What kind of build was this?

Daily projects can be completed alone or by pair programming. We were encouraged to reach out to our Team Lead or other students for help as needed. We could not receive help with them. They reinforce the day's lesson, provide practice for the material, and expand your knowledge through self learning.

What did I do?

I utilized knex migrations to design and write a schema for the cars table according to specifications. I configured knex to connect to a database using sqlite3. I wrote endpoints to suppors CREATE and READ operations. I used Postman to test my API.
