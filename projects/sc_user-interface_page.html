What is this?

Having the ability to create user interfaces is important for Full-Stack Web Developers. This project tested our abilities with Semantic HTML, display types, CSS specificity, flex box, and basic usage of git.

What kind of build was this?

Sprint Challenges are projects we completed alone within a 3 hours. We could not receive help with them. They act as tests of our knowledge and skill regarding the week's learning.

What did I do?

I built the header and navigation with HTML and CSS. I styled the boxes with css according to the prescribed colors. I created an about page according to the design file. I added animations with CSS.
