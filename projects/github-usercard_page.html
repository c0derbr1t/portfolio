Project: GitHub Usercards

Unit 2 - Sprint 1   |   December 5, 2019

See the Project!

GitHub Repo

What is this?

This is a daily project where we were given instructions on GitHub to follow. The purpose of this assignment is "Request data from the Github API and create a component based on the data you get back."

What kind of build was this?

Daily projects can be completed alone or by pair programming. We were encouraged to reach out to our Team Lead or other students for help as needed. They reinforce the day's lesson, provide practice for the material, and expand your knowledge through self learning.

What did I do?

I used axios to request data from the GitHub API. After studying the data received from it, I created the index.js file and added in dynamic data from the API. I made a component based on my profile, and added it to the DOM. Then I added some of my followers and added cards for them, also. I completed a stretch goal by integrating each person's GitHub contributions graph.
