Project: HTTP Movies

Unit 3 - Sprint 3   |   January 30, 2020

See the Project!

GitHub Repo

What is this?

This is a daily project where we were given instructions on GitHub to follow. The purpose was to React Router and Redux to give a list of movies update and delete functionality.

This project improves upon a project that we did during Unit 2 - Sprint 3. We built out a lot of the already existing functionality for this project. You can view the GitHub for that project here.

What kind of build was this?

Daily projects can be completed alone or by pair programming. We were encouraged to reach out to our Team Lead or other students for help as needed. We could not receive help with them. They reinforce the day's lesson, provide practice for the material, and expand your knowledge through self learning.

What did I do?

This project came with a server for us to use to get and store information, and a built out app structure and content. I added a route to update a movie, and created the component with form to update with a PUT request, then re-render the movie card. Then I added a delete button that made a DELETE request, and then re-route the user to the main movie list.
